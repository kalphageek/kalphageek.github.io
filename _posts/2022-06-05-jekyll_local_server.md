```bash
$ cd kalphageek.github.io/

# Buidle. 작업이 끝난 결과물을 실제 서버나 gitHub에 업로드, 배포할 경우에는 다음 명령어로 빌드 한 후에 _site에 생성된 html 등을 사용하시면 됩니다.
$ bundle exec jekyll build

# Start
$ bundle exec jekyll serve
```
